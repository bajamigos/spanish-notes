SIELE S4 Oral Exam Overview

Volver a Recursos

Purpose: The SIELE S4 exam tests oral Spanish proficiency (15-20 minutes, 5 tasks, 250 points max), targeting CEFR levels A1-C1. The student aims for C1 (215-250 points), requiring fluent, precise speech on complex topics.

Tasks:

  • Task 1 (A1): Answer 4 personal questions (e.g., daily life, experiences). 2 min.
  • Task 2 (A2): Describe a photo of an everyday scene. 2-3 min.
  • Task 3 (B1): Respond to 2 scenarios (personal, public, e.g., requesting info). 3-4 min.
  • Task 4 (B2): Answer 3 questions on a topic text (80-120 words, e.g., opinions on issues). 4-5 min.
  • Task 5 (C1): Monologue arguing for/against a statement, with examples and conclusion. 4-5 min.

Scoring Criteria (by SIELE examiners):

  • Language Use (0-5): Fluency, cohesion, grammar (e.g., subjunctive), vocabulary, idioms. C1 needs near-error-free, fluid speech.
  • Fulfillment (0-5): Completing all task points with detail, examples, and context fit. C1 requires structured arguments.

Tutor Role: Practice all tasks, emphasizing Task 5 for C1. Use recordings, theme-based prompts, and feedback on fluency, grammar, and task completion. Access sample prompts at siele.org.
